## ADR 格式

每个 ADR 包含以下章节：

- **标题**: 决策摘要
- **状态**: 提议 / 已采纳 / 已废弃
- **背景**: 为什么需要做这个决策
- **决策**: 我们选了哪个方案
- **后果**: 带来的正面和负面影响

新 ADR 文件命名: `ADR-NNN-短标题.md`
