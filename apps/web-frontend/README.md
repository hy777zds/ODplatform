# apps/web-frontend — Web 前端（占位）

> **状态**: 占位，V1.1 启动开发

本子项目将提供 ODPlatform 的 Web 前端界面，用于：

- 数据集管理（上传、预览、标注）
- 训练任务配置与监控
- 推理结果可视化

## 为什么现在是空的？

V1.0 聚焦核心引擎（platform），前端在 V1.1 启动。预留此目录确保：

- 前后端协作边界在 Day 0 就已划定
- 共享 schemas（`packages/shared-schemas/`）前后端共用一份
